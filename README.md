### 二、对Java的理解
- java是一门面向对象的编程语言。
- java语言与平台无关，不用担心操作系统的变化，利用虚拟机实现了“一次写成，处处运行”的效果。
- java内置多线程可以同时完成多个任务。
- java具有很强的安全性，不允许访问计算机的其它部分.
